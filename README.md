# PlayStation-Midi-Pedal
A simple hack into the playstation pedal for driving games. The result is an assignable midi signal (0-128) controlled with the foot.

The build is extremely simple: locate the potentiometer in the pedal, in my case it was a 50k, but it should make no difference what size its resistance is. connect it following the schematics:

<img width="540" height="260" alt="circuit (1)" src="https://github.com/user-attachments/assets/e87e43a6-0b2d-4fc4-9c72-85105e61bf85" />

Use ur midi connector of choice, i used a TRS ring since it adapted well to my case and made the device very compact.
Remember, you will need for sure a way to power your arduino.
